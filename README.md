## Project Overview

Secure File Locker is a Python-based cryptography project developed as part of the BIT 4148 – Advanced Cryptography course. The application demonstrates the practical implementation of symmetric key cryptography by enabling users to securely encrypt and decrypt files.

The system uses the Fernet implementation of AES encryption provided by Python's Cryptography library to protect file contents from unauthorized access.

---

## Objectives

- Demonstrate the use of symmetric encryption techniques.
- Implement secure file encryption and decryption.
- Explore cryptographic key generation and management.
- Understand the role of cryptography in data protection.

---

## Features

- Secure key generation
- File encryption using AES-based Fernet encryption
- File decryption using the generated key
- User-friendly command-line interface
- Protection of sensitive file contents

---

## Technologies Used

- Python 3
- Cryptography Library (Fernet)
- Command Line Interface (CLI)
